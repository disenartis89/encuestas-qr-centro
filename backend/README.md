# Backend

Lógica de negocio del proyecto EncuestasQR (en Java).

Archivos:

- `Usuarios.java` — registro de alumnos y profesores.
- `Encuestas.java` — creación y gestión de encuestas (pendiente terminar).

Para el QR todavía no hay archivo. Se creará en una rama `feature/qr` durante las prácticas.

## Nota sobre IntelliJ

Para que IntelliJ lo reconozca como proyecto Java, al abrir la carpeta `encuestas-qr-centro` pulsa `File` → `Project Structure` → `Modules` y marca `backend` como carpeta de fuentes (Sources). Si tu proyecto real usa Maven o Gradle, tu compañero de back ya tendrá configurado un `pom.xml` o `build.gradle`; aquí lo dejamos simple para centrarnos en Git.
