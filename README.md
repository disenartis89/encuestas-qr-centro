# EncuestasQR — Centro Educativo (mini-proyecto de práctica)

Aplicación ficticia para registrar **profesores** y **alumnos**, con un panel de **administrador** y un sistema de **QR para encuestas** del centro educativo.

Este proyecto NO es para ejecutar de verdad: es un campo de pruebas para practicar Git simulando un equipo con tres áreas:

- `backend/` — lógica y API (el equipo de back).
- `frontend/` — interfaz web (el equipo de front).
- `bbdd/` — base de datos (el equipo de BBDD).

## Cómo usarlo

Abre el archivo `GUIA_GIT.md` en la raíz del proyecto y sigue las prácticas en orden. Cada práctica te hace tocar una parte distinta del proyecto, simulando lo que harías con tus compañeros.
